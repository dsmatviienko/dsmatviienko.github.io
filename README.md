# dariamatviienko.com

Personal academic website, hosted on GitHub Pages.

## Before publishing — edit checklist
- [ ] `index.html`: replace every block marked `<!-- EDIT -->` (bio, paper titles, abstracts, teaching, ballroom section, email)
- [ ] Add `photo.jpg` (your portrait, ~600–1000px wide, JPG)
- [ ] Add `cv.pdf`
- [ ] Create a `papers/` folder and add `paper1.pdf`, `paper2.pdf`, etc. (or link to SSRN instead)
- [ ] Update the Google Scholar link (or remove it)

## File structure
```
index.html    ← the whole site (design + content in one file)
CNAME         ← tells GitHub Pages your custom domain
photo.jpg     ← your photo (you add this)
cv.pdf        ← your CV (you add this)
papers/       ← your paper PDFs (you add this)
```
